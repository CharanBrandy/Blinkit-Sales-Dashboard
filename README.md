# Blinkit Sales Analysis Dashboard (Power BI)

## Project Overview
This Power BI dashboard analyzes Blinkit sales performance, outlet trends, item types, and customer ratings to generate business insights.

## Key Insights
- Total Sales: $1.2M
- Average Sales: $141
- 8523 items analyzed
- Avg Rating: 3.9

## Dashboard Features
- Sales trend by outlet establishment year
- Fat content & item type analysis
- Outlet size & location performance
- Interactive filters and visuals

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization
